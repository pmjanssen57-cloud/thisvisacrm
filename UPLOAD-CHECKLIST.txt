THiS CRM v0.9.1 upload checklist

1. Preserve the netlify/ folder.
2. Preserve the public/ folder.
3. Upload/commit all files from this ZIP to the GitHub repo.
4. Confirm Netlify environment variables remain set:
   - NETLIFY_DATABASE_URL
   - CRM_ACCESS_TOKEN (temporary fallback until removed)
5. Deploy through Netlify.
6. Confirm the main desktop menu appears on one line and the Library tab is visible.
7. Confirm mobile bottom navigation still works.
8. Test Library, Clients, Calendar and Tools after deploy.
