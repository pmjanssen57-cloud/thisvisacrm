THiS CRM v0.10.3 upload checklist

1. Preserve the netlify/ folder.
2. Preserve the public/ folder.
3. Upload/commit all files from this ZIP to the GitHub repo.
4. Confirm Netlify environment variables remain set:
   - NETLIFY_DATABASE_URL
   - CRM_ACCESS_TOKEN (temporary fallback until removed)
5. Deploy through Netlify with Clear cache and deploy if possible.
6. Confirm adviser login still works.
7. Confirm /portal opens the client portal login screen.
8. In one client record, enable Portal, generate an access code, publish an update and test the client portal login.
9. Confirm only client-facing portal information appears.
10. From the client portal, submit one adviser note and one client planning note.
11. Confirm the Dashboard shows a new client portal note and the Client Portal section can mark it reviewed.
12. Test Weather, Timezone, Currency and Calculator tools in the client portal.
13. Test Clients, Calendar, Library, Tools and Print Profile after deploy.
